Práctica taller 1: Arduino digital y analógico

Salida digital: parpadeo de un led (Blink)​

  • Realizar el montaje de la figura. Rled = 220 ohm.
  • Dibujar en el cuaderno el esquema eléctrico.
  • Cargar en la placa arduino un programa que haga parpader el led cada dos .
  • Modificar el programa anterior para que el led parpadee cada medio segundo.
  • Modificar el programa de forma que el led envíe un S.O.S de forma contínua.

blink.ino
void setup() {
  pinMode(13, OUTPUT);
}
void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Entrada digital: Encender unled con un pulsador​

  • Realizar el montaje de la figura. Rled = 220 ohm y Rpulsador = 10 kOhm.
  • Dibujar en el cuaderno el esquema eléctrico.
  • Cargar en la placa arduino un programa como el facilitado que haga que se encienda el led al apretar el pulsador y que se apague al soltarlo.
  • Modificar el programa anterior introduciendo variables para determinar los pines de entrada y salida.

Para ello, basta con poner al inicio del programa (antes de void setup()) la definición de la variable:

int LedPin = 13;

Después, en lugar de poner 13 se pondría LedPin. Esto ahorra tiempo cuando tenemos que hacer cambios de pines.

pulsador.ino
void setup() {
  pinMode(3, INPUT);
  pinMode(13, OUTPUT);
}
void loop() {
  if (digitalRead(3) == HIGH){
    digitalWrite(13, HIGH);
  }
  else {
  digitalWrite(13, LOW);
  }
}

Entrada analógica: Sensor de luz con LDR​

  1. Realizar el montaje de la figura. RLDR = 10 kΩ.
  2. Dibujar en el cuaderno el correspondiente esquema eléctrico.
  3. Cargar en la placa arduino el programa indicado (listado de programa 3.1)
  4. Comprueba si, efectivamente, el led se enciende al oscurecer la LDR.
  5. Abre el monitor serie y observa los valores que se muestran en pantalla.
  6. Graba con el programa SimpleScreenRecorder la pantalla del monitor serie en la que se vea cómo cambian los valores en ambiente claro y oscuro.

ldr.ino
void setup() {
  pinMode(13, OUTPUT);
  Serial.begin(9600);
}
void loop() {
  int ldrValue = analogRead(A0);
  Serial.println(ldrValue);
  if(ldrValue < 200){
    digitalWrite(13, LOW);
  }
  else{
    digitalWrite(13, HIGH);
  }
}

Salida analógica: Potenciómetro​

  1. Realiza el montaje de la figura.
  2. Dibuja en el cuaderno su esquema eléctrico.
  3. Cargar en la placa arduino el programa indicado.
  4. Observa cómo es la iluminación del led según giras el cursor del potenciómetro.
  5. Abre el monitor serie y haz una grabación en la que se vea lo del punto 4 y los valores del monitor serie.
potenciometro.ino
// Entradas analógicas: 0-1023
// Salidas analógicas: 0-255
int LedLight = 0;
void setup() {
  pinMode(3, OUTPUT);
  Serial.begin(9600);
}
void loop() {
  int potValue = analogRead(A3);
  LedLight = map(potValue,0,1023,0,255);
  analogWrite(3,LedLight);
  Serial.print("potValue = ");
  Serial.print(potValue);
  Serial.print(" LedLight = ");
  Serial.println(LedLight);
  delay(10);
}
